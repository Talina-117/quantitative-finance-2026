# quantitative-finance-2026

量化交易学习项目，包含股票数据获取、技术指标计算与交互式可视化分析报告。

## 📊 在线可视化报告

项目通过 GitHub Pages 部署了股票分析报告：

👉 **[兆易创新(603986.SH) 综合分析报告](https://talina-117.github.io/quantitative-finance-2026/)**

### 报告内容

**核心数据卡片**
- 最新收盘价、PE(TTM)、PB、年涨幅、股息率等关键指标一览

**基本面分析**
- 公司概况、盈利能力指标、估值指标、成长性指标、财务健康度表格

**技术面分析**
- 移动平均线（MA5/MA10/MA20/MA60）趋势判断
- MACD 指标分析（金叉/死叉信号）
- RSI 相对强弱指标（超买/超卖判断）
- KDJ 随机指标分析

**近13个月日线走势图**
- SVG 绘制的收盘价走势曲线（2025年7月—2026年6月）

**多维度综合评分**
- 基本面质量（25%权重）、技术面信号（25%权重）、资金面强度（20%权重）、估值合理性（30%权重）
- 综合评分雷达图与评分依据详解

**风险提示**
- 高估值风险、短期超买、跌停信号等风险因素汇总

**数据来源与说明**
- Tushare Pro 数据接口说明与指标参数标注
