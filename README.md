# quantitative-finance-2026

量化交易学习项目，包含股票数据获取、技术指标计算与可视化分析。

## 📊 在线可视化报告

项目部署了交互式股票分析报告，基于 ECharts 实现数据可视化：

👉 **[风华高科(000636.SZ) 全面分析报告](https://talina-117.github.io/quantitative-finance-2026/)**

报告包含以下内容：

### 公司概况
- 风华高科基本信息、行业定位、公司属性

### 核心指标一览
- 最新收盘价、PE(TTM)、PB、年涨幅、股息率等关键指标卡片

### 七大分析模块（含 7 个交互式图表）

1. **价格走势与K线分析** — 日K线走势图（支持缩放、拖拽交互）
2. **技术面分析** — RSI + MACD 组合指标图
3. **筹码与资金分析** — 资金流向图（超大单/大单/中单/小单净流入）
4. **股东分析** — 股东户数变化趋势图
5. **基本面分析** — 营收与利润趋势图
6. **估值评估** — 分红派息历史图
7. **多维度评估雷达** — 综合评分雷达图（估值/成长/盈利/筹码/风险）

### 风险提示
- 高估值风险、一字跌停信号、筹码分散等风险因素汇总
