# 基本概念解释：K线、基本面、技术面

## 一、K线（Candlestick Chart）

### 1.1 什么是 K 线？

K 线（又称蜡烛图，Candlestick Chart）起源于日本江户时代的米市交易，由本间宗久发明，是目前全球金融市场最常用的价格图表形式。一根 K 线记录了一定时间周期内（如 1 分钟、1 日、1 周等）的四个关键价格：

- **开盘价（Open）**：该周期内的第一笔成交价
- **收盘价（Close）**：该周期内的最后一笔成交价
- **最高价（High）**：该周期内的最高成交价
- **最低价（Low）**：该周期内的最低成交价

### 1.2 K 线的构成

```
        最高价 ──→  ┌────┐  ← 上影线
                    │    │
        收盘价 ──→  │████│  ← 实体（阳线，收盘 > 开盘）
                    │████│
        开盘价 ──→  │████│
                    │    │
        最低价 ──→  └────┘  ← 下影线
```

- **实体**：开盘价与收盘价之间的矩形区域
  - 阳线（红色/白色）：收盘价 > 开盘价，表示价格上涨
  - 阴线（绿色/黑色）：收盘价 < 开盘价，表示价格下跌
- **影线**：实体上下方的细线
  - 上影线：最高价到实体上沿
  - 下影线：最低价到实体下沿

> 注：中国股市惯例为 **红涨绿跌**，与欧美市场相反。

### 1.3 K 线形态的含义

| 形态 | 特征 | 含义 |
|------|------|------|
| 大阳线 | 长红实体，短影线 | 强烈看涨信号 |
| 大阴线 | 长绿实体，短影线 | 强烈看跌信号 |
| 十字星 | 开盘≈收盘，上下影线长 | 多空均衡，可能反转 |
| 锤子线 | 小实体，长下影线 | 底部反转信号 |
| 倒锤子 | 小实体，长上影线 | 顶部反转信号 |
| 吞没形态 | 后一根K线实体完全包住前一根 | 趋势反转信号 |

### 1.4 K 线的时间周期

- **日内**：1分钟、5分钟、15分钟、30分钟、60分钟
- **日线**：每日一根（最常用）
- **周线/月线**：用于中长期趋势判断

## 二、基本面分析（Fundamental Analysis）

### 2.1 定义

基本面分析是通过研究影响资产内在价值的经济、财务、行业等因素，判断资产是否被**高估或低估**的分析方法。其核心假设是：**市场价格最终会回归到内在价值**。

### 2.2 分析维度

#### 宏观经济层面
- GDP 增长率、CPI/PPI 通胀指标
- 货币政策（利率、存款准备金率）
- 财政政策（税收、政府支出）
- 国际经济环境

#### 行业层面
- 行业生命周期（导入期→成长期→成熟期→衰退期）
- 行业竞争格局（波特五力模型）
- 产业政策与监管环境
- 上下游产业链关系

#### 公司层面
- **财务报表分析**：资产负债表、利润表、现金流量表
- **盈利能力指标**：ROE（净资产收益率）、ROA（总资产收益率）、毛利率、净利率
- **成长性指标**：营收增长率、净利润增长率
- **估值指标**：PE（市盈率）、PB（市净率）、PS（市销率）、PEG
- **偿债能力**：资产负债率、流动比率、速动比率
- **公司治理**：管理层质量、股权结构

### 2.3 适用场景

- **长期投资**：价值投资、成长投资
- **行业配置**：选择景气度向上的行业
- **风险识别**：排除财务造假、经营恶化的公司

### 2.4 局限性

- 信息可能滞后（财报按季度发布）
- 估值具有主观性（不同分析师给出不同目标价）
- 短期价格波动常与基本面脱节
- 无法精确择时

## 三、技术面分析（Technical Analysis）

### 3.1 定义

技术面分析是通过研究**历史价格、成交量等市场交易数据**，来预测未来价格走势的分析方法。其三大核心假设：

1. **市场行为包容消化一切**：所有已知信息已反映在价格中
2. **价格以趋势方式演变**：趋势一旦形成，更可能延续而非反转
3. **历史会重演**：市场参与者的行为模式具有重复性

### 3.2 主要分析工具

#### 趋势分析
- **移动平均线（MA）**：平滑价格序列，识别趋势方向
  - 短期（5日/10日）、中期（20日/60日）、长期（120日/250日）
  - 金叉（短期上穿长期）→ 看涨信号
  - 死叉（短期下穿长期）→ 看跌信号
- **趋势线**：连接高点或低点，判断支撑与阻力

#### 震荡指标
| 指标 | 用途 | 超买/超卖区间 |
|------|------|-------------|
| RSI（相对强弱指标） | 衡量价格变动速度 | >70 超买，<30 超卖 |
| MACD | 趋势跟踪与动量 | 信号线交叉判断买卖 |
| KDJ | 随机指标 | >80 超买，<20 超卖 |
| 布林带（Bollinger Bands） | 波动率与价格区间 | 触及上轨/下轨判断 |

#### 成交量分析
- 量价配合：价涨量增（健康上涨）、价跌量缩（正常回调）
- 量价背离：价涨量缩（上涨乏力）、价跌量增（恐慌抛售）

#### 形态分析
- **反转形态**：头肩顶/底、双顶/底（M顶/W底）、圆弧顶/底
- **持续形态**：旗形、三角形、矩形整理

### 3.3 适用场景

- **短期交易**：择时、波段操作
- **入场/出场点位选择**
- **趋势确认与反转预警**

### 3.4 局限性

- 基于历史数据，无法预测突发事件
- 信号有时滞性
- 存在大量虚假信号（需要多重确认）
- 过度拟合风险（参数优化后回测很好，实盘失效）

## 四、基本面 vs 技术面：对比总结

| 维度 | 基本面分析 | 技术面分析 |
|------|-----------|-----------|
| 核心逻辑 | 价格回归内在价值 | 价格反映一切信息 |
| 分析对象 | 财务数据、宏观经济 | 价格、成交量、技术指标 |
| 时间框架 | 中长期（数月到数年） | 短期（数天到数周） |
| 目标 | 判断「买什么」 | 判断「何时买/卖」 |
| 优势 | 发现被低估的优质资产 | 精准择时、快速反应 |
| 劣势 | 择时能力弱 | 无法判断资产内在价值 |
| 代表人物 | 巴菲特、格雷厄姆 | 利弗莫尔、约翰·墨菲 |

### 最佳实践：两者结合

多数成功的投资者将基本面与技术面结合使用：

1. **用基本面选股**：筛选出财务健康、成长性好的公司
2. **用技术面择时**：在技术指标发出买入信号时入场
3. **用基本面止损**：当公司基本面恶化时离场
4. **用技术面风控**：通过支撑位、均线等设置止损位
